# 세종도서관 해외정책자료 수집 시스템 — 개발문서

---

## 1. 시스템 개요

해외 16개국 100+ 정부·연구기관의 정책문서를 자동 크롤링하여 **필터링 → 점수 산출 → 균등 선별 → 메타데이터 보강 → 검증 → Excel 납품**까지 처리하는 파이프라인이다.

**목표 산출물**: 2026년 발행 해외 정책문서 300건 (세종도서관 납품 Excel 양식)

## 4. CLI 진입점 (main.py)

### 4.1 서브커맨드

```bash
python main.py summary                          # 사이트 현황 조회
python main.py crawl-uk [--site CODE|--all]      # 영국 GOV.UK
python main.py crawl-us [--site CODE|--all]      # 미국
python main.py crawl-se [--site CODE|--all]      # 스웨덴
python main.py crawl-sg [--site CODE|--all]      # 싱가포르
python main.py crawl COUNTRY [--site|--all]      # 범용 크롤러
```

### 4.2 공통 인자

| 인자 | 기본값 | 설명 |
|------|--------|------|
| `--site CODE` | - | 특정 사이트 코드 (예: Z00057) |
| `--all` | False | 해당 국가 전체 사이트 크롤링 |
| `--max N` | 100 | 사이트당 최대 수집 건수 |
| `--data-dir PATH` | None | 데이터 저장 경로 (기본: ./data/) |
| `--api-key KEY` | DEMO_KEY | GovInfo API 키 (미국용) |
| `--force` | False | current_use=X 사이트도 크롤링 |
| `--list` | False | 사이트 목록만 출력 |

@dataclass

    no: int                  # 일련번호
    code: str                # 기관코드 (Z00001)
    name: str                # 영문 기관명
    name_prev: str           # 수정전 기관명
    country_code: str        # 국가코드 (GB, US, NO 등)
    country: str             # 국가 (한글)
    org_type: str            # 유형 (정부기관, 연구기관)
    name_kr: str             # 한글 기관명
    name_kr_prev: str        # 수정전 한글 기관명
    acronym: str             # 약어
    url: str                 # 수집 URL
    url_prev: str            # 수정전 URL
    note: str                # 비고
    brm_category: str        # BRM 대분류 (예: "교육", "환경, 기후")
    expected_count: str      # 연간 예상 수량
    exclude: str             # 수집제외 사유
    alias1: str              # 기관명 기타1
    alias2: str              # 기관명 기타2
    alias3: str              # 기관명 기타3
    collected_2025: str      # 2025년 수집건수

    tags: list = field(default_factory=list)

### 6.2 BaseSiteCrawler (추상 기반 클래스)

```python
class BaseSiteCrawler(ABC):
    # 초기화
    def __init__(data_dir: Path)
        # requests.Session 생성, User-Agent 설정

    # 추상 메서드
    @abstractmethod
    def crawl(site: Site, max_results: int = 100) -> list[dict]

    # HTTP 요청 (3회 재시도, SSL/Playwright 폴백)
    def _get(url, max_retries=3) -> Response | None

    # BeautifulSoup 파싱 (Cloudflare 감지 → Playwright 폴백)
    def _soup(url) -> BeautifulSoup | None

    # 표준 문서 딕셔너리 생성
    def _doc(site, **fields) -> dict

    # 결과 JSON 저장
    def save(site, results) -> Path | None

    # Playwright 서브프로세스 실행 (JS 렌더링 폴백)
    @staticmethod
    def _pw_get(url, wait_ms=5000) -> Response | None

    # Cloudflare 챌린지 감지
    @staticmethod
    def _is_cloudflare_challenge(soup) -> bool

    # PDF 직접 링크 제거
    @staticmethod
    def _filter_pdf_direct_links(results) -> list[dict]
```

### 6.3 재시도·복원력 로직

```
HTTP 요청 실패 시:
  ├─ 429 (Rate Limit)
  │   └─ Retry-After 헤더 대기, 없으면 2^n × 5초 대기
  ├─ 403/503
  │   └─ Playwright JS 렌더링 폴백
  ├─ SSL 오류
  │   └─ verify=False로 재시도
  ├─ Timeout / ConnectionError
  │   └─ 2^n초 대기 후 재시도
  └─ Cloudflare "Just a Moment"
      └─ Playwright 폴백

최대 재시도: 3회
요청 간격: 1.0초 (영국), 1.5초 (미국/기타)
```

def filter_by_relevance(results, min_score=0.0) -> list[dict]:
