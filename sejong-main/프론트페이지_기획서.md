## 1. 페이지 설계 개요

### 1.1 설계 방향
- **싱글 페이지 대시보드** — 파이프라인 전체 현황을 한 눈에 파악
- **운영자 중심 UI** — 크롤링 실행, 모니터링, 데이터 확인, Excel 다운로드까지 원스톱
- **데이터 시각화** — 국가별/단계별 진행률을 직관적으로 표현

### B. 파이프라인 진행 상태 (Pipeline Status)

5단계 파이프라인을 시각적 스텝퍼(Stepper)로 표현하고, 실행 제어 버튼과 실시간 로그를 제공한다.

#### B.1 스텝퍼 (Pipeline Stepper)

```
[1.크롤링] ──▶ [2.선별] ──▶ [3.보강] ──▶ [4.검증] ──▶ [5.출력]
```

| 단계 | 라벨 | 실행 스크립트 | 세부 단계 |
|------|------|------------|---------|
| 1 | 크롤링 | `main.py crawl-*` + 추가 크롤러 | US, SE, SG, 범용 31개국, 추가소스 (**GB 이번 라운드 제외**) |
| 2 | 선별 | `select_300.py` | 3-Pass 알고리즘 (키워드 최소 → 국가 쿼터 → 잔여 충원) + 영어 필터 |
| 3 | 보강 | `resolve_dates_v2.py` → `resolve_pdfs.py` → `resolve_authors.py` → `resolve_excerpts.py` → `enrich_selection.py` → `metadata_resolver.py` | 날짜 → PDF → 저자(최대5인) → 요약 → 키워드/잡지/권호 → ISBN/ISSN/유형/CCL |
| 4 | 검증 | `validate_selection.py` | 3중 전수 검증: 날짜(2026+) → PDF(Content-Type+100KB) → 언어(영어) |
| 5 | 출력 | `export_selection.py` | 25컬럼 Excel (맑은 고딕, 헤더 #2F5496) |

**단계별 상태 표시:**

| 상태 | 아이콘 | 색상 | 설명 |
|------|-------|------|------|
| 대기 | `⏳` | `gray-400` | 아직 실행 안 됨 |
| 진행중 | `🔄` (회전 애니메이션) | `blue-500` | 현재 실행 중 |
| 완료 | `✅` | `green-500` | 성공 완료 |
| 실패 | `❌` | `red-500` | 에러 발생 |
| 부분완료 | `⚠️` | `yellow-500` | 일부 사이트 실패 |

**진행중 상태 확장:**


#### B.2 실행 제어 버튼

|------|------|-----|------|

### C. 국가별 수집 현황 (Country Distribution)

수평 막대 차트로 국가별 수집 건수를 표시한다.

**데이터:**
```
국가코드 → 국기 이모지 + 국가명 + 수집 건수 + 선별 건수
```

| 필드 | 설명 |
|------|------|
| 국가코드 | sites.csv의 country_code |
| 수집 건수 | 해당 국가 data/*.json 내 전체 문서 수 |
| 선별 건수 | selection_400.json 내 해당 국가 문서 수 |
| 쿼터 | select_300.py의 country_quota |

**표시 형식:**
```
🇺🇸 US  ████████████░░░  487건 (선별 170/170)
         ↑ 수집 전체      ↑ 괄호: 선별/쿼터
```

**인터랙션:**
- 막대 클릭 → 하단 테이블이 해당 국가로 필터링
- 호버 → 툴팁: 기관 수, 평균 점수, 수집일

**정렬:** 수집 건수 내림차순 (상위 8개국 표시, "더보기" 클릭 시 전체 35개국)

**특수 표시:**
- GB: 회색 처리 + "이번 라운드 제외" 라벨
- 차단 사이트 보유 국가: 경고 아이콘 (IEA, CBO, OECD, IMF, WEF, NBER, CIGI, RAND — 8개 기관 403 차단)
